HMDB-Traverser Version 1: A tool for matching mass spectrometry data to metabolites in the Human Metabolome Database (HMDB).

Features:
LC-MS Search Metabolite Matching
    
Instructions for Use:
  1) To customize the search parameters, first open the HMDB_config.py file.
  2) Update the "option" numerical argument within the XPATH values in the LOCATOR dictionary for the desired parameter.
     * Note: The numerical argument corresponds to the option’s position in HMDB’s dropdown menu.
  3) Once parameters are set, run HMDB_main.py to automatically match m/z inputs to metabolites.
